# Contributing

Thanks for your interest.

We welcome:
- Bug reports
- Security reports (private, see SECURITY.md)
- Suggestions and feature requests

Pull requests:
- We may decline PRs unless requested first
- Please open an issue before starting any large change

Bug reports should include:
- WordPress version
- WooCommerce version
- Plugin version
- Whether you are using Blocks checkout
- Steps to reproduce
- Relevant logs or a redacted Support Bundle
